# Catch the Pokemon - Setup & Run Instructions:

# 1. Clone the Repository
git clone https://github.com/avinashv-dev/catch-the-pokemon.git
cd catch-the-pokemon

# 2. Install Dependencies
npm install

# 3. Run the Game
npm run dev


# Technologies Used
- React (Vite)
- Tailwind CSS
- PokeAPI
- React Context API
